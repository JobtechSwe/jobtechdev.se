Taxonomy

JobTech Taxonomy innehåller data som vanligtvis används inom arbetsmarknaden och Arbetsförmedlingen.

Datan består av strukturerade listor, som är relevanta för matchning av lediga jobb och arbetssökande.

JobTech Search

JobTech Search är en sökmotor för alla nuvarande jobbannonser från Platsbanken.

Målet är att skapa ett enkelt men ändå mångsidigt API som ska passa alla som bygger alla slags applikationer som innehåller data från jobbannonser.

Job Stream

Job Stream API är för er som bygger applikationer och behöver förvara alla annonser lokalt.

Med Job Stream kan du hämta alla förändringar i annonserna en gång/minut. Nya, borttagna eller uppdaterade annonser.
